02_목록 관련 태그

ul(Unordered List) 태그 : 순서 없는 목록을 작성하는 태그(영역)

li(List Item) 태그 : 목록에 들어가는 내용을 작성하는 태그

  • 김밥
  • 서브웨이
  • 백반
  • 햄버거

ol(Ordered List)태그 : 순서가 있는 목록을 작성하는 태그(영역)

            기본값 : 1부터 1씩 증가하는 숫자

            *ol 태그의 type 속성

            type = "a" --> 영어 소문자 순서
            type = "A" --> 영어 대문자 순서
            type = "i" --> 로마 소문자 순서
            type = "I" --> 로마 대문자 순서

            + 추가 속성
            start = "5" --> 순서가 5부터 시작

            reversed --> 순서 반대로
  1. 김밥
  2. 서브웨이
  3. 백반
  4. 햄버거
